### **Рекуррентные формулы для q-биномиальных коэффициентов**

#### **1. Основная рекуррента (аналог треугольника Паскаля)**
\[
\binom{n}{k}_q = \binom{n-1}{k}_q + q^{n-k} \cdot \binom{n-1}{k-1}_q
\]
